# Data Pipelines: ETL vs ELT vs EtLT

## ETL (Extract, Transform, Load)
Data is cleaned before being stored.
Used in industries like banking and healthcare.

## ELT (Extract, Load, Transform)
Data is stored first, then cleaned later.
Used in modern cloud systems.

## EtLT (Hybrid)
Combination of both ETL and ELT.

## Legal Compliance
 ETL is safer for sensitive data
ELT needs strong security
Important factors:
 Data privacy
 Access control
